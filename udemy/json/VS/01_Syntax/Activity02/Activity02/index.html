Seven rules in building JSON data

  1. Data is written inside curly braces.
  2. Data is represented as key value pairs
  3. Keys should be written in dbl quotes.
  4. Key and value are separated by a colon(:).
  5. Key value pairs are separated by a comma(,).
  6. Array data is held inside square brackets.
  7. Object data is held inside curly braces.
